# Сайт готельно-туристичного комплексу «П'яте Колесо»

Вітаємо! Цей репозиторій містить вихідний код офіційного сайту ГТК «П'яте Колесо» (с. Лазещина, Карпати).

## 🚀 Огляд проекту
Сайт розроблений для представлення готельних послуг, демонстрації номерного фонду, можливостей відпочинку та спрощення процесу бронювання для гостей.

## 🛠 Технології
Проект створений за допомогою чистих веб-технологій, що забезпечує високу швидкість завантаження та легкість у підтримці:
* **HTML5**: Структура сторінки.
* **CSS3 (Flexbox/Grid)**: Сучасний адаптивний дизайн, що коректно відображається на смартфонах, планшетах та десктопах.
* **JavaScript (ES6)**: Інтерактивні елементи, галереї номерів, робота модальних вікон та анімації появи елементів при скролі.
* **Google Fonts**: Підключені шрифти *Cormorant Garamond* та *Inter*.

## 📂 Структура файлів
* `index.html` — головний файл сайту, що містить всю розмітку, стилі та скрипти.
* `*.jpg` — файли зображень, що використовуються в дизайні (фото готелю, номерів, природи).

## ⚙️ Інструкція з налаштування

### 1. Редагування контенту
Ви можете редагувати тексти та інформацію безпосередньо у файлі `index.html`:
* Щоб змінити текст, знайдіть відповідний тег (наприклад, `<h1>`, `<p>` або назви номерів).
* Кольори та брендові відтінки можна налаштувати у блоці `:root` всередині `<style>`.

### 2. Оновлення фотографій
Сайт використовує зображення з локальної папки. 
* Щоб замінити фото, просто замініть відповідний файл (наприклад, `1.jpg`, `3.jpg`) на новий файл з такою ж назвою.
* Переконайтеся, що нові файли мають такий же формат та приблизно такий же розмір для збереження швидкості сайту.

### 3. Форма бронювання
Форма налаштована на відправку даних методом `GET` на адресу: `http://pyate-koleso.com.ua/booking-hotel-or-cottage`. 
* *Примітка:* Якщо ви бажаєте змінити спосіб отримання заявок (наприклад, на власну пошту), вам потрібно буде підключити сервіс обробки форм (наприклад, Formspree або власну PHP-скрипт/сервер).

## 📱 Адаптивність
Сайт повністю адаптивний. Якщо ви додаєте нові секції, використовуйте існуючі медіа-запити (`@media`) у кінці блоку стилів для коректного відображення на різних пристроях.
